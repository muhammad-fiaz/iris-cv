# Security Policy

## Supported Versions

We support security updates for the following versions:

| Version | Supported |
|---------|-----------|
| 0.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing the maintainer at `contact@muhammadfiaz.com`. 

We will investigate the issue and attempt to resolve it as quickly as possible. Please do not report security vulnerabilities via public GitHub issues.
